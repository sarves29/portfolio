# Django Portfolio - Sarvesvara Ramar

Generated scaffold for a Django portfolio website. Replace secrets and configure email in settings.py before production.

## Quick start
1. python -m venv venv
2. source venv/bin/activate
3. pip install -r requirements.txt
4. python manage.py migrate
5. python manage.py createsuperuser
6. python manage.py loaddata initial_projects.json
7. python manage.py runserver
